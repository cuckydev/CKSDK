Detailed Description

template<bool Grad, bool Quad, bool Tex, bool Semi, bool Raw>
struct CKSDK::GPU::Poly< Grad, Quad, Tex, Semi, Raw >

Poly is a template for a polygon primitive.

Template Parameters
Gradtrue if each vertex has a color
Quadtrue if the polygon is a quad
Textrue if textured
Semitrue if semi-transparency is enabled
Rawtrue if modulation is disabled

The structure will contain v0 through v2, or v0 through v3 if Quad is true.

Each vertex will contain xy, as well as uv (if Tex is true), and c (if Grad is true).

The color of the polygon, if Grad is false, is set by v0.c.
